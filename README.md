# Orbis — AI Intelligence Layer for Maritime Security

> *Protect oceans. Stop illegal fishing. Keep seas sustainable.*

---

## The Problem

Every year, **26 million tons** of fish are illegally harvested from the world's oceans. Illegal, unreported, and unregulated (IUU) fishing devastates marine ecosystems, threatens food security for billions, and costs the global economy **$23 billion annually**. Existing monitoring systems are slow, fragmented, and overwhelmed. By the time authorities respond — the vessel is gone.

**The ocean is 361 million square kilometres. It has never had an intelligence layer. Until now.**

---

## The Solution — Orbis

Orbis is an AI-powered maritime intelligence platform that monitors vessel activity in real time, detects illegal operations, enforces regulations autonomously, and protects marine ecosystems at global scale.

Think of it as the **command center for ocean law enforcement** — part satellite intelligence, part AI legal agent, part Coast Guard co-pilot.

---

## Key Features

### Real-Time Vessel Intelligence Dashboard
A live operational view of all maritime activity. Filter by date, region, or threat level. Every suspicious incident is surfaced instantly — no noise, just signal.

### 3D Interactive Globe
A stunning, interactive 3D globe renders vessel positions worldwide. **Green markers** indicate registered, compliant vessels. **Red markers** flag unregistered or suspicious vessels operating outside legal boundaries. Heat signatures reveal clusters of illegal activity before enforcement teams even know to look.

### AI Enforcement Agent
Click any incident on the globe and Orbis deploys an AI enforcement agent instantly. Powered by **Exa**, it retrieves the exact maritime laws applicable to that vessel's GPS coordinates. The **Gemini API** then generates a Coast Guard-style enforcement response with full legal reasoning — ready to act in seconds, not hours.

### Intelligence Console
A deep-dive investigation tool for tracking specific vessels. Orbis analyzes historical behavior patterns, cross-references past incident reports, and identifies repeat offenders — building an intelligence dossier that makes prosecution airtight.

### Reporting & Analytics Dashboard
Performance metrics, incident statistics, and trend analysis — all in one place. Generate AI-authored reports across any custom date range. Evidence-grade documentation with one click.

### Clearance-Based Access Control
Orbis is built for real operational security:
- **Public Trust** — Standard access for monitoring and reporting
- **Secret** — Restricted access for active enforcement operations
- **Top Secret** — Full system access with the ability to assign roles, manage permissions, and control classified intelligence

Authentication and authorization are handled by **Zero** — enterprise-grade, battle-tested security from day one.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-org/orbis.git
cd orbis

# Install dependencies
npm install

# Configure environment
cp .env.example .env.local
# Add your API keys: VITE_BASE44_APP_ID, Gemini, Exa, Zero

# Run locally
npm run dev
```
